Этапы нашей работы

1. Брифинг заказчика

Вы заполняете анкету, помечаете пожелания по визуальному оформлению сайта. Указываете важные блоки сайта, информацию про вашу целевую аудиторию

2. Составление прототипа

Разработка схематичного прототипа изображения сайта. Блочные структуры сайта, наполненные текстом и изображениями. Отправка прототипа на согласование с клиентом

3. Разработка дизайна

Создание дизайн сайта: добавление шрифтов, цветовых сочетаний, графических изображений и т.д. Отправка готового дизайна на согласование с клиентом

4. Разработка адаптива

Создание адаптивных версий сайта для различных устройств (если требуется)

5. Верстка сайта

Верстка полученного от дизайнера макета сайта, используя HTML + CSS
